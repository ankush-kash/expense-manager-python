# Expense Manager

A simple command-line Expense Manager built using Python that helps users track daily expenses, categorize spending, and save records permanently using JSON.

---

## Features

- Add new expenses
- View all saved expenses
- Show expense summary by category
- Save expenses automatically in a JSON file
- Simple and beginner-friendly CLI interface

---

## Built With

- Python 
- JSON for data storage
